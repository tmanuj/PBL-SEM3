# PBL-SEM3
STEPS, HOW TO RUN CODE IN YOUR SYSTEM
1. MAKE A FOLDER AND PLACE ALL FILES IN THE MAIN VERSIONS INSIDE IT.
2. YOUR VS CODE S SETUP MUST BE PROPER , WITH C AND OTHERS.
3. IN TERMINAL RUN THIS COMMAND(COPY IT)
   gcc gtk_app.c backend.c -o food_manager `pkg-config --cflags --libs gtk+-3.0`

4. THEN RUN THIS COMMAND
  ./food_manager

5. IT WILL RUN GTK AND POP UP PROJECT WINDOW INTERFACE.
6. PROVIDE INFORMATIONA AND DETAILS, RELATED TO PERSONS IDENTITY AND HIS ORDER
7. IN CASE OF VERIFICATION OF HIS IDENTIY
8. CODE WILL VERIFY HIS/HER PHONE NO WITH THE PHONE NO PRESENT IN THE EXISTING VIP.TXT
9. IF IT MATCHES THEN WORKING OF PRIORITY QUEUE WILL TAKE PLACE, ELSE NORMAL ORDER QUEUE.

10. FOR PERSONS ORDER IT WILL STORE IN THE FILE DATA.TXT.
